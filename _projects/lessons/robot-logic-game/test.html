Robot Logic Game — Dev Test

Pick a level below to launch it in a fullscreen overlay. Press ✕ in the corner to return.

How to play

  • Drag blocks from the Blocks panel into the Program panel.
  • Drag program blocks up or down to reorder them — drop into a Repeat or If body to nest.
  • Click ✕ on a program block to remove it.
  • Press Run to execute your program.
  • Press Reset to clear your program and move the robot back to start.
